# rastreamento-entregas-logistica
Logistica para rastreamento de entregas

Baixe o projeto fazendo:
```
git clone https://github.com/bruno-santana/rastreamento-entregas-logistica.git
```
ou
```
git clone git@github.com:bruno-santana/rastreamento-entregas-logistica.git
```

Entre na pasta
```
cd rastreamento-entregas-logistica
```

E faça o comando
```
make up
```

Depois disso, faça
```
make setup
```

Caso não possua o Make no seu PC, abra o arquivo "makefile" na raiz do projeto e execute os passos correspondentes ao up e setup. 

A aplicação estará respondendo no endereço: <http://localhost:8080>
